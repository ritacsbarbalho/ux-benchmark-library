# Catálogo Benchmark

Repositório de screenshots e referências de páginas de catálogo de produtos concorrentes para análise de UX e produto.

Objetivo:
Analisar padrões de listagem de produtos utilizados por diferentes produtos digitais e identificar boas práticas, oportunidades e diferenças na organização, navegação e visualização de itens no catálogo.

Estrutura:

produto/
  imagens da página de catálogo

Exemplo:

hympar/
  hympar-catalogo-geral.png
  hympar-catalogo-filtros.png

mercado-farma/
  mef-catalogo-geral.png
  mef-catalogo-filtros.png
