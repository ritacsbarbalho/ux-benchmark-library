# Cadastro Benchmark

Repositório de screenshots e referências de fluxos de cadastro de produtos concorrentes para análise de UX e produto.

Objetivo:
Analisar padrões de cadastro utilizados por diferentes produtos digitais e identificar boas práticas, oportunidades e diferenças de fluxo.

Estrutura:

produto/
  imagens do fluxo de cadastro

Exemplo:

hympar/
  hympar-cadastro-step1.png
  hympar-cadastro-step2.png

mercado-farma/
  mef-cadastro-step1.png
