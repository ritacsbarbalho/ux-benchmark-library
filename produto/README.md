# Produto Benchmark

Repositório de screenshots e referências de páginas de produto de produtos concorrentes para análise de UX e produto.

Objetivo:
Analisar padrões de páginas de produto utilizados por diferentes produtos digitais e identificar boas práticas, oportunidades e diferenças na apresentação de informações, como preço, descrição, estoque, distribuidores e ações disponíveis ao usuário.

Estrutura:

produto/
  imagens da página de produto

Exemplo:

hympar/
  hympar-produto-geral.png
  hympar-produto-detalhes.png

  mef-produto-geral.png
  mef-produto-detalhes.png
