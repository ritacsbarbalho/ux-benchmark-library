# Promoções Benchmark

Repositório de screenshots e referências de mecânicas promocionais de produtos concorrentes para análise de UX e produto.

Objetivo:
Analisar padrões de apresentação e aplicação de promoções utilizados por diferentes produtos digitais e identificar boas práticas, oportunidades e diferenças na comunicação de descontos, benefícios e regras promocionais.

Estrutura:

produto/
  imagens das páginas ou componentes de promoções

Exemplo:

  hympar-promocoes-detalhes.png
