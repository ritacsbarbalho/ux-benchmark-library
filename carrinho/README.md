# Carrinho Benchmark

Repositório de screenshots e referências de páginas de carrinho de produtos concorrentes para análise de UX e produto.

Objetivo:
Analisar padrões de carrinho de compra utilizados por diferentes produtos digitais e identificar boas práticas, oportunidades e diferenças na organização dos itens, cálculo de valores, aplicação de promoções e ações disponíveis ao usuário.

Estrutura:

produto/
  imagens da página de carrinho

Exemplo:

hympar/
  hympar-carrinho-geral.png
  hympar-carrinho-valores.png

  mef-carrinho-geral.png
  mef-carrinho-descontos.png
